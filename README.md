🏠 Azubi-Apartments-Project

Ein Webprojekt, mit dem Benutzer Wohnungen suchen, mieten, kaufen oder verkaufen können.

📌 Projektbeschreibung

Azubi-Apartments-Project ist eine einfache Immobilien-Webanwendung, die es Nutzern ermöglicht:

🏡 Wohnungen zu durchsuchen

🔍 Immobilien zu finden (Miete / Kauf)

💰 Wohnungen zum Verkauf oder zur Vermietung anzubieten

📍 Details wie Preis und Standort anzuzeigen


Das Projekt basiert auf HTML, CSS, JavaScript und Firebase.

🚀 Features

Anzeige von Wohnungen aus einer Datenbank (Firebase Firestore)

Dynamisches Laden der Daten in die Website

Mehrere Seiten (z. B. Home, Apartments)

Responsive Design (für Handy & Desktop)

Einfache Navigation zwischen Seiten


🧰 Verwendete Technologien

HTML5

CSS3

JavaScript (Vanilla JS)

Firebase (Firestore + Analytics)
